# LeadPredictor

A lightweight lead forecasting calculator built with HTML, CSS, and JavaScript.

## Features

- Interactive lead and prospect sliders
- Responsive dashboard layout
- Dynamic chart and statistics updates
- Language and currency controls

## How to run

Open `index.html` in a browser, or serve the folder using a local web server.

## Notes

This project is designed for quick prototyping and front-end experimentation.
